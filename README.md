# trucoit-com

Hugo static site for [trucoit.com](trucoit.com), deployed via GitHub Pages.

- [Contributing](#contributing)
- [Code of conduct](#code-of-conduct)
- [License](#license)

## Build

```bash
cd site-trucoit-com/
make install  # Install Hugo
make build    # Build site
make test     # Local server
```
